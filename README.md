# Just a Sec iOS

**Course project:** iOS Mobile Application  
**Description:** An iOS mobile application demonstrating UI design, data handling, and core iOS development concepts using Swift and UIKit.

## 📌 Overview

Just a Sec is a mobile app built during the Mobile Apps iOS course.  
The project showcases key iOS development skills, including user interface design with storyboards, data handling, and navigation between screens using UIKit and Cocoa Touch components.

## 📱 Features

- User interface for setting and viewing alarms  
- Alarm triggers only while the user is on the Alarm screen  
- Snooze functionality implemented for 5 minutes  
- App design following iOS Human Interface Guidelines

## 🛠 Technologies Used

- Xcode  
- Swift  
- UIKit / Cocoa Touch Framework  
- Storyboards & Auto Layout  
- iOS Simulator for testing

## 🎯 Key Takeaways

- Implementing view controllers with lifecycle management  
- Building responsive and user-friendly mobile interfaces  
- Navigating between screens using storyboards  
- Applying software engineering principles to iOS mobile app development  
- Understanding the end-to-end flow of iOS applications from UI to functionality

## ⚠️ Note

The alarm functionality currently only triggers while the user is viewing the Alarm screen. Snooze is implemented for 5 minutes only; other customised snooze options are not implemented due to time constraints for the course deadline. Additional snooze features may be added in future iterations.

The API key previously used for Firebase integration has been removed from this repository for security purposes. The project is functional for demonstration and learning purposes.
