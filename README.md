# Just a Sec iOS

**Course project:** iOS Mobile Application  
**Description:** An iOS mobile application demonstrating UI design, data handling, and core iOS development concepts using Swift and UIKit.

## 📌 Overview

Just a Sec is a mobile app built during the Mobile Apps iOS course.  
The project showcases key iOS development skills, including user interface design with storyboards, data handling, and navigation between screens using UIKit and Cocoa Touch components.
